# Agent OS

Open-source management system for AI agent teams. Not just orchestration — operational philosophy included.

## What is Agent OS?

Agent OS is the system we built to run 4 AI agents 24/7 building a real company. It solves the problems nobody talks about:

- **Agents coast when you stop watching.** We built an output clock that detects idle agents in real-time.
- **Context degrades after 4 hours.** We built shift boundaries with automatic retros and fresh context restarts.
- **85% of agent output should be rejected.** We built quality gates and peer review meetings.
- **Agents run out of ideas.** We built a dual-loop ideation system that generates work, not just executes it.
- **Agents need meetings too.** We built structured async meetings with mandatory challenges.
